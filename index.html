There's this great site, TypeGreek.com, that converts Latin characters into precomposed polytonic Greek Unicode. I use it all the time! But for years, no one has put together a parallel site for polytonic Hebrew. I found this so irritating that I finally broke down and created one myself. This is that site.

The Hebrew font you see is SBL Hebrew, hosted locally using Font Squirrel's brilliant webfont generator. The rest of the text is set in Raph Levien's Inconsolata, served via Google Fonts. I'm also indebted to ..., without which I couldn't have put this site together, and to TypeGreek.com for inspiring this site.
